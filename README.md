รายงานสรุปอุณหภูมิและความชื้นของ **ห้องเก็บสีย้อม** (สี Disperse / สีย้อมเส้นด้าย)
จากเซนเซอร์ **Inkbird IBS-TH3-PLUS** ที่บันทึกอัตโนมัติทุก 10 นาที

> ทำไมต้องสนใจความชื้น: สีย้อมผง โดยเฉพาะสี Disperse ไวต่อความชื้นและความร้อนมาก
> ความชื้นสูงทำให้สี **จับตัวเป็นก้อน · เฉดเพี้ยน · กระจายตัวไม่ดี · ขึ้นรา**
> เกณฑ์ที่เหมาะกับการเก็บสีคือ **ความชื้นสัมพัทธ์ต่ำกว่า ~60%** และเก็บในที่เย็น

## 📊 ดูรายงาน

เปิดไฟล์ [`index.html`](index.html) ด้วยเบราว์เซอร์ได้เลย — เป็นไฟล์เดียวจบ
(ฝังข้อมูลและกราฟ Chart.js ไว้ในไฟล์แล้ว เปิดได้แม้ออฟไลน์ เหมาะกับการดูบนมือถือ)

หรือเปิดผ่านเซิร์ฟเวอร์ภายในเครื่อง:
```bash
python3 -m http.server 8910
# แล้วเปิด http://127.0.0.1:8910/index.html
```

## 🛠️ สร้างรายงานใหม่เมื่อมีข้อมูลเดือนถัดไป

```bash
# 1) วางไฟล์ CSV ใหม่จากเซนเซอร์ (UTF-16, คั่นด้วย Tab) แล้วแก้ชื่อไฟล์ใน parse.py
python3 parse.py            # อ่าน CSV -> data.json (+ สรุปสถิติ)

# 2) ฝังข้อมูล + Chart.js ลงใน index.html
python3 build.py            # template.html + data.json + chartjs.min.js -> index.html
```

## 📁 โครงสร้างไฟล์

| ไฟล์ | หน้าที่ |
|---|---|
| `index.html` | **รายงานหน้าเดียวจบ** (ผลลัพธ์ที่ใช้เปิดดู) |
| `template.html` | เทมเพลตหน้าเว็บ (มี `__DATA_PLACEHOLDER__`) |
| `parse.py` | อ่าน CSV → คำนวณสถิติ → `data.json` |
| `build.py` | ฝัง `data.json` + `chartjs.min.js` ลงเทมเพลต → `index.html` |
| `data.json` | ข้อมูลที่ประมวลผลแล้ว |
| `chartjs.min.js` | ไลบรารีกราฟ (Chart.js v4) |
| `IBS-TH3-PLUS_*.csv` | ข้อมูลดิบจากเซนเซอร์ |
